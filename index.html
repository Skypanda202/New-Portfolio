Aspect-Based Sentiment Analysis (GAN-BERT)

I developed an Aspect-Based Sentiment Analysis (ABSA) model by integrating BERT with Generative Adversarial Networks (GANs) to analyze restaurant reviews with high precision. Leveraging semi-supervised learning, the model effectively utilized both labeled and unlabeled data, enabling robust sentiment classification across five key aspects. The solution achieved an impressive 96.9% accuracy, outperforming baseline BERT (90%) and GAN (91%) models. The project was implemented using Python, TensorFlow, Hugging Face Transformers, Pandas, NumPy, Scikit-learn, and Matplotlib, with detailed performance metrics, training visualizations, and confusion matrix analysis for comprehensive evaluation.
